# Estimate Builder

> **Category:** Cost & Estimation
> **Priority:** P1 - Post-Launch
> **Status:** Brainstorming

## Overview

Build detailed cost estimates from takeoff quantities by assigning cost items from the cost database. Supports line-item estimates with material, labor, and equipment breakdowns, subtotals by trade/division, and summary views for proposal preparation.
